# Icon Wizard

Source: https://docs.fontawesome.com/web/add-icons/icon-wizard

The Icon Wizard is a tool for modifying existing Font Awesome icons to create customized versions.

## Features

- Modify existing Font Awesome icons visually
- Combine and adjust icon elements
- Create custom variations without design software
- Export modified icons for use in projects

## Requirements

- Font Awesome Pro plan for full access
- A Kit configured for custom icon uploads

## Usage

Access the Icon Wizard through your Font Awesome account to:
1. Select a base icon
2. Apply modifications (resize, reposition, combine)
3. Export the customized icon
4. Upload to your Kit for use in projects
