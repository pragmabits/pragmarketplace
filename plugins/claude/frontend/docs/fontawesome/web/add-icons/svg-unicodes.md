# SVG Unicodes

Source: https://docs.fontawesome.com/web/add-icons/svg-unicodes

Reference icons by their Unicode values when using the SVG+JS framework.

## Usage

Each Font Awesome icon has a unique Unicode value that can be used to reference it:

```html
<i class="fa-solid" data-icon="&#xf007;"></i>
```

## Finding Unicode Values

Unicode values for each icon are available on the Font Awesome icon detail pages and in the metadata files included with downloads.

## When to Use

- When HTML markup needs to reference icons by stable identifiers
- When working with systems that strip class-based icon references
- When building tools that programmatically generate icon references
