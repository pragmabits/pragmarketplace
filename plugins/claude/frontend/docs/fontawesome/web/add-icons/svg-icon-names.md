# SVG Icon Names

Source: https://docs.fontawesome.com/web/add-icons/svg-icon-names

Reference icons by their names when using the SVG+JS framework.

## Usage

Icons can be referenced by name attribute when using the SVG+JS framework:

```html
<i data-icon="user" class="fa-solid"></i>
```

## Notes

- Icon names follow the `fa-` prefix convention (without the prefix in the data attribute)
- Names can be found on icon detail pages at fontawesome.com
- Aliases are supported — both old and new names work
- This method works alongside class-based and Unicode-based referencing
