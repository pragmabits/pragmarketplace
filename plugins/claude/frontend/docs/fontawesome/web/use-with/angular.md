# Angular Integration

Source: https://docs.fontawesome.com/web/use-with/angular

Font Awesome's official Angular component for integrating icons into Angular projects.

## Key Requirements

- Plan to use the SVG+JS method of Font Awesome with Angular
- Have project files ready for modification

## Official Resources

1. **API Documentation**: Available on npm at `@fortawesome/angular-fontawesome`
2. **GitHub Repository**: For issues and contributions

## Alternative Approaches

Developers can also opt to use Web Fonts with CSS with a Kit or by self-hosting.

The component is distributed via npm, following standard Node.js package management practices for Angular projects. The component works with both Angular and AngularJS.
