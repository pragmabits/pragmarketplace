# Browser Support

Source: https://docs.fontawesome.com/web/dig-deeper/browser-support

Font Awesome supports and works out-of-the-box in the following web browsers.

## Supported Browsers

| Browser | Desktop | Mobile |
|---------|---------|--------|
| Chrome, Edge (Blink) | Last 2 major versions and enterprise versions | iOS latest and Android 4+ |
| Firefox (Gecko) | Last 2 major and ESR versions | iOS latest and Android 4+ |
| Safari (WebKit) | Last 2 major versions | iOS 9+ |

## Unsupported Browsers

Internet Explorer receives no support, though Microsoft Edge is fully supported.

## Notes

- Other Blink and Gecko-based browsers likely work as well.
- For browsers not explicitly listed experiencing issues, users should contact the Font Awesome or respective browser communities.
- Users encountering bugs in supported browsers are encouraged to submit issues on GitHub.
- Font Awesome works with multiple implementation methods including Web Fonts, SVGs, hosted Kits, and self-hosted solutions, all contributing to broad compatibility.
