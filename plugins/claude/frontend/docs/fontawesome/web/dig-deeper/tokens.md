# Tokens

Source: https://docs.fontawesome.com/web/dig-deeper/tokens

Font Awesome Tokens let you securely use Font Awesome in third-party apps and environments, whether that's WordPress, Figma, a CLI, or a plugin.

## Token Types

### 1. Package Manager Tokens

Enable installation of Font Awesome Pro/Pro+ icons using npm, yarn, or pip.

### 2. API Tokens

Facilitate access to icons, Kit data, and account information across various applications.

## Key Benefits

- Access Free or Pro/Pro+ icons and metadata
- Access your Kits data, including custom icons
- Ability to refresh/replace a token if the one you're using gets compromised

## Security Considerations

Tokens provide access to your account — keep your token safe just like you do for your account login. If compromised, generate a new token from your Font Awesome Account dashboard.

## Practical Applications

API tokens support integration with:

- **WordPress** — Read Kits Data
- **Figma** — Read Kits Data plus icon metadata access
- **Custom applications** — Requiring Pro icon access

## Management

Tokens are managed through the Font Awesome Account under the dedicated Tokens section.
