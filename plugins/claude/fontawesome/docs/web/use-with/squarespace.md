# Squarespace Integration

Source: https://docs.fontawesome.com/web/use-with/squarespace

## Add Kit to Header

Settings > Advanced > Code Injection. Paste:

```html
<script src="https://kit.fontawesome.com/YOUR-KIT-ID.js" crossorigin="anonymous"></script>
```

## Add Icons

Create code blocks, use `<i>` tags:

```html
<i class="fa-solid fa-camera-retro"></i>
<i class="fa-thin fa-camera-retro"></i>
<i class="fa-duotone fa-camera-retro"></i>
<i class="fa-sharp fa-solid fa-camera-retro"></i>
```
