# Python and Django

Source: https://docs.fontawesome.com/web/use-with/python-django

> **Note:** The official Python package is **deprecated**. Recommendation: use a Kit.

## Setup

Copy Kit embed code into HTML `<head>`:

```html
<script src="https://kit.fontawesome.com/YOUR-KIT-ID.js" crossorigin="anonymous"></script>
```

## Self-Hosting

Also supported: download Kit and follow Web Fonts or SVG+JS self-hosting instructions.
