# Flip Icons

Source: https://docs.fontawesome.com/web/style/flip

Static flip classes mirror the icon along an axis.

## Flip Classes

| Class | Effect |
|-------|--------|
| `fa-flip-horizontal` | Mirrors horizontally (left-right) |
| `fa-flip-vertical` | Mirrors vertically (top-bottom) |
| `fa-flip-both` | Mirrors both horizontally and vertically |

## Examples

```html
<i class="fa-solid fa-snowboarding"></i>
<i class="fa-solid fa-snowboarding fa-flip-horizontal"></i>
<i class="fa-solid fa-snowboarding fa-flip-vertical"></i>
<i class="fa-solid fa-snowboarding fa-flip-both"></i>
```

> **Note:** `fa-flip-both` is equivalent to `fa-rotate-180` visually but uses scale transforms instead of rotation.
