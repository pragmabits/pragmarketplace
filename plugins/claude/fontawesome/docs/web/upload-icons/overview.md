# Upload Your Own Icons

Source: https://docs.fontawesome.com/web/add-icons/upload-icons

A Pro Plan with a perpetual license is necessary to use this feature. Users need an active Pro-level plan with access to v7, and a kit configured to use Pro icons.

## Process Overview

1. **Design Your Icon** — Follow icon design guidelines for grid systems, shapes, and design methodology.
2. **Prepare Icons for Upload** — Meet technical requirements for SVG files before uploading.
3. **Upload to a Kit** — Upload SVG files to Pro Kit through "Uploaded Icons" section using drag and drop, file browsing, or direct SVG paste (compatible with Adobe Illustrator exports). The system applies best practices automatically during processing.

## Key Constraints

- Icons bind to a specific Kit and function only in projects referencing that Kit
- Web use only currently
- Upload limits depend on user's plan
- Custom icons can share names with official Font Awesome icons (differentiated by prefix)

## Usage in Projects

| Style | Prefix | Example |
|-------|--------|---------|
| Kit Custom Icons | `fa-kit` | `<i class="fa-kit fa-basketball-hoop-broken"></i>` |
| Kit Custom Duotone Icons | `fa-kit-duotone` | `<i class="fa-kit-duotone fa-basketball-hoop-broken"></i>` |

Custom icons support all standard Font Awesome styling options including sizing, coloring, rotating, transforms, masking, and layering.

## Kit Downloads

Starting with v6.4, users can download kits containing uploaded icons in web font or SVG formats for self-hosting, or for desktop applications. The kit version must be set to "Latest 7.x" or at least 6.4.
