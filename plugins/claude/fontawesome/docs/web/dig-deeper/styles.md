# Icon Packs, Families, and Styles

Source: https://docs.fontawesome.com/web/dig-deeper/styles

## Key Definitions

- **Packs**: A group of one or more icon families that share a cohesive visual personality
- **Families**: A group of related icon styles that share the same basic design but vary in weight/style
- **Styles**: A consistent and specific visual weight
