# Quick Start

Source: https://docs.fontawesome.com/web/setup/get-started

This page serves as the entry point for developers wanting to rapidly integrate Font Awesome icons into web projects.

## Core Setup Process — Three Steps

### Step 1: Project Setup

Font Awesome Kits are the easiest way to get Font Awesome icons into your projects. Just create a new Font Awesome Kit and add the Kit embed code to the `head` of each template or page.

### Step 2: Icon Discovery and Integration

Users browse thousands of available icons and implement them via HTML using `<i>` tags with CSS classes specifying style and icon name.

### Step 3: Styling Enhancement

Apply additional styling options including sizing, animations, duotone effects, icon lists, stacking, counters, and power transforms.

## Available Icon Styles

- Solid
- Regular
- Light (Pro)
- Thin (Pro)
- Duotone (Pro)
- Sharp variants: solid, regular, light, thin (Pro)
- Sharp Duotone (Pro)
- Brand icons (Free)

## Troubleshooting

Verify kit embed code placement; consult troubleshooting documentation for additional support.
