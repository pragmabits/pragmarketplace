# shadcn/ui Installation Overview

## Quick Start

```bash
pnpm dlx shadcn@latest init
```

This guided setup handles framework selection, dependency installation, and project structure configuration automatically.

## Framework Options

shadcn/ui supports multiple React-based frameworks:

- **Next.js** — Full-stack React framework with server components (`-t next`)
- **Vite** — Lightning-fast frontend build tool (`-t vite`)
- **TanStack Start** — Full-stack type-safe framework (`-t start`)
- **Laravel** — PHP framework with React integration (`-t laravel`)
- **React Router** — Client-side routing for React (`-t react-router`)
- **Astro** — Static site generator with React support (`-t astro`)

## Initialization Process

1. **Framework Selection** — Choose your preferred development environment
2. **Icon Library** — Select icon collection matching your design needs
3. **Theme Configuration** — Customize colors and design tokens
4. **Project Structure** — Automatic setup of required files and folders

## Next Steps

After initialization, explore:

- **components.json** — Core configuration file for your project
- **Theming Documentation** — Customize appearance with CSS variables
- **Dark Mode Setup** — Add light/dark theme support
- **CLI Documentation** — Use commands to add components on-demand
